# go 数组

## 数据结构

- `len`
- `type`

## 创建 初始化 访问

```go
var arr [5]int;
arr := [5]int;
arr := [5]int{10, 20, 30, 40, 50};
arr := [...]int{10, 20, 30, 40, 50};
arr := [5]string{ 99: "foo" } // 100个元素
arr := [...]*int{new(int), new(int)}
```

## 赋值

同长度，同类型的数组之间，可以赋值

```go
a1 := [3]int{1,2,3};
var a2 [3]int;
a2 = a1
```

## 多维数组

```go
arr := [4][2]int{
    {1,2}, {2,2}, {1,1}, {2,1},
}
```
