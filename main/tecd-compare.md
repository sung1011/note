## etcd vs redis

|          | etcd                         | redis                            |
| -------- | ---------------------------- | -------------------------------- |
| 一致性   | raft保证强一致性             | 不保障一致性                     |
| 数据模型 | kv                           | kv,zset,list,hash等              |
| 事务     | 强 一个操作中改变多个key     | 弱 不回滚,失败后后续操作仍会执行 |
| 读写性能 | 读写性能一般                 | 读写性能极高(内存)               |
| 存储     | 分布式一致性存储             | 内存数据库                       |
| 场景     | 关键的系统配置和服务发现信息 | 缓存和非关键数据存储             |
| 续期     | keepAlive机制                | expire手动                       |
