
- `函数调用栈` 函数代码栈

```code
main() {
    a = 1, b = 2
    rs = add(a, b)
}

add(x, y) {
    var sum int
    sum = x + y
    return sum
}
```

```stack
x = 1
y = 2
sum = 3
...
rs = 0
a = 1
b = 2
```

- `最小栈` O(1)获取栈的最小元素

```code
# 额外保存最小栈；队列同理。

stack = [10, 12, 9, 14]; // min = stack[2] = 9
minsStackIndex = [0, 2];

# push
stack = [10, 12, 9, 14, 4]; // min = stack[4] = 4
minsStackIndex = [0, 2, 4]; // push时发现4是最小栈，则将其索引值同步push到额外栈中。

# pop pop pop
stack = [10, 12]; // min = stack[0] = 10
minsStackIndex = [0]; // pop时同步pop
```
