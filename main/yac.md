# yac

## 特性

- 让PHP进程之间**共享**一些简单的数据
- 高效地缓存一些页面结果

## 限制

- 缓存的键长度不能超过48字节，太长的话可以md5结果后再使用
- Value的最大长度不能超过64M，压缩后的长度不能超过1M
- 当内存不够的时候, Yac会有比较明显的踢出率 (所以如果要使用Yac, 那么尽量多给点内存...)
- 最多可用存储32768个不同的Cache值，如果需要更多，需修改yac.keys_memory_size
- Yac的生命周期为PHP-FPM生命周期，Cli模式场景暂时未找到合适方案处理缓存
