# 二叉搜索树 binary search tree / 二叉排序树 Binary Sorted Tree

    基于二叉树, 各节点值不同且`有序` (左 < 根 < 右)

## 结构

```js
                27
             /      \
           14         35
          /  \       /  \
        10   19     31   42
```

> [二叉树](ds-binary-tree.md) 每个节点 <= 2个子节点.  

## 特征

- `二分法查找` O(log n)
